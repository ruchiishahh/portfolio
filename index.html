S.T.O.P. Domestic Violence - IOS Application

This IOS Application was created with my team at the first ever XX Hacks (2017), an all female hackathon for high school students in the Bay Area. Our mobile application focused on spreading awareness about domestic violence and sexual violence and attempted to prevent this imperative issue in today's world. Some of the apps features include calling for help in the case of an emergency, tips about what to do in domestic violence situations, and information about the issue to raise awareness. The app won 2nd in the category of Advancing Social and Civil Rights.

No Stress Dress - IOS Application

This IOS Application was created with my team at Pixel Hacks 2018. No Stress Dress allows users to not only have an app pick out their outfits, but also plan out outfits for the week, plan outfits for travel, wear rarely worn clothes more often, and online shop while adding fantasy items to a virtual closet. Users are able to either have the app pick an outfit for them depending on their desired style of the day, the weather, and even their mood, in order to expedite their morning routine or even try out a new style using clothes that are already in the userâ€™s wardrobe. Secondly, users are able to plan outfits about a week in advance whether it be to make the packing process simpler for a vacation or just to keep an extremely busy week from becoming a full-blown catastrophe. The app won Best Presentation Overall.

Safe Haven - IOS Application

This IOS Application was created at XX Hacks 2018. Some of the features of this app are that it will allow the user to see the reported cases of sexual violence in the area and can report any cases they see. You are also notified if you walk past a reported location. We also make information regarding safety and precautions listed on the app with links to helpful sources. The app can also actively call 911, the Sheriff department, and your listed emergency contact. This app won 2nd place overall.

Ants

This is one of the projects I created in my UC Berkeley CS 61A class. In this project, I created a game where bees attacked ants and the each type of ant had different ways to protect and defend themselves from the bees. This project was developed with Python and extensively used object oriented programming as each subclass inherited from its parent/superclass.

Scheme

I created this project in my UC Berkeley CS 61A class as well. Here, I created an interpreter that used Python to interpret the user's input given in scheme and then used the 3-function infinite loop of Read-Eval-Print to analyze the code lexically and parse it into a python function that ended up outputing a value. In this project, I used both Python and Scheme built in functions, special forms, and primitive to convert Scheme code using Python implementation.

ImageLab

This is a project I worked on in my AP Computer Science A class at Cupertino High School. I utilized the features of Java to open a file on my desktop which was a picture and then coded multiple filters to modify the image by manipulating each and every pixel of the file. Some filters I created were HorizontallyFlipped, Mirror, Black&WhiteFilter, DoubleImage and many more. This project was developed through Java and focused on opening and modifying image files.

Minesweeper/Pacman

These project was created in my AP Computer Science A class in Cupertino High School. I developed two games that are quite popular today. Minesweeper and Pacman were both created using Java and extensively used the Model View Controller pattern to logically separate my code into three components. Additionally, I also created a GUI for my games to create an enhanced User Interface.
